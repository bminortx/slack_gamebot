# Slack Gamebot

A gamebot for slack.  Supports

* Leaderboards
* Stats
* Smack Talking

Written in python2/django 1.9

# How to:

1. On command line:

```
cd /path/to/repo
echo API_TOKEN = "<YOUR-API-KEY>" > slackbot_settings.py
./manage.py migrate
./manage.py run_bot
```

2. Invite your gamebot to channel.
3. Start interacting with commands like `gamebot help`.
